
## Tecnologías Utilizadas

- Next.js 14
- Material-UI

## Instalación

1. Clonar el repositorio: `git clone https://github.com/YAFcod3/maquetado-adobe-dfl.git`
2. Instalar las dependencias: `npm install`

## Uso
```bash
npm run dev
```
